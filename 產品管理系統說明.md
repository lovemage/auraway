### 🚀 使用方法

#### 1. 啟動後端服務器
```bash
cd backend
npm start
```

#### 2. 初始化產品數據（首次使用）
```bash
cd backend
npm run init-products
```

#### 3. 訪問後台管理
開啟瀏覽器訪問：`http://localhost:5001/admin.html`

#### 4. 產品管理操作
1. 點擊側邊欄「商品管理」
2. 可以查看所有產品列表
3. 點擊「新增產品」添加新商品
4. 點擊「編輯」修改現有商品
5. 點擊「上架/下架」切換商品狀態
6. 在庫存警告區塊快速補貨

### 📊 API端點

#### 產品相關API
- `GET /api/products` - 獲取所有產品
- `GET /api/products/active` - 獲取上架產品
- `GET /api/products/category/:category` - 按分類獲取產品
- `GET /api/products/:id` - 獲取單一產品
- `POST /api/products` - 新增產品
- `PUT /api/products/:id` - 更新產品
- `PATCH /api/products/:id/stock` - 更新庫存
- `PATCH /api/products/:id/toggle` - 切換上架狀態
- `DELETE /api/products/:id` - 刪除產品

### 🎨 產品數據結構

```javascript
{
  name: "產品名稱",
  description: "產品描述",
  price: 1280,              // 售價
  originalPrice: 2000,      // 原價（可選）
  stock: 50,                // 庫存數量
  category: "健康保健",      // 產品分類
  badge: "熱銷",            // 產品標籤
  images: [                 // 產品圖片陣列
    "/images/產品資料夾/圖片1.jpg",
    "/images/產品資料夾/圖片2.jpg"
  ],
  specifications: {         // 產品規格
    content: "18g x 20包/盒",
    storage: "常溫陰涼乾燥處保存",
    origin: "台灣製造",
    ingredients: "主要成分說明",
    usage: "食用方法",
    features: ["特色1", "特色2", "特色3"]
  },
  tags: ["標籤1", "標籤2"],
  isActive: true,           // 是否上架
  sortOrder: 1              // 排序順序
}
```
